## Usage

Install

    go get github.com/erniebrodeur/goprompt/cmd/goprompt

then

    goprompt

## Shell Integration

### Zsh

```
#!/bin/zsh
#------------------------------
# Prompt
#------------------------------
function precmd () {

}
```
